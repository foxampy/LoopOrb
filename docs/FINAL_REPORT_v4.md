## 🔧 Требуемые API Endpoints

### Пользователи
- `GET /api/user/me` — Данные пользователя
- `PATCH /api/user/me` — Обновление профиля
- `GET /api/user/[id]` — Профиль другого пользователя

### Проекты
- `GET /api/projects` — Список проектов
- `GET /api/projects/[id]` — Детали проекта
- `POST /api/projects/[id]/invest` — Инвестирование
- `POST /api/projects/[id]/like` — Лайк
- `POST /api/projects/[id]/comment` — Комментарий

### Миссии
- `GET /api/missions` — Список миссий
- `GET /api/user/missions` — Миссии пользователя
- `POST /api/missions/[id]/complete` — Завершение миссии

### Достижения
- `GET /api/achievements` — Список достижений
- `GET /api/user/achievements` — Достижения пользователя
- `GET /api/achievements/leaderboard` — Таблица лидеров

### Feed
- `GET /api/feed` — Лента новостей
- `POST /api/posts` — Создание поста
- `POST /api/posts/[id]/like` — Лайк
- `POST /api/posts/[id]/comment` — Комментарий

### DAO
- `GET /api/dao/proposals` — Список предложений
- `POST /api/dao/proposals` — Создание предложения
- `POST /api/dao/vote` — Голосование
- `GET /api/dao/treasury` — Казначейство

### Глобус
- `GET /api/globe/markers` — Маркеры объектов
- `GET /api/sensors/data` — Данные сенсоров
- `GET /api/objects` — Водные объекты

### VOD-Lab
- `GET /api/vod-lab/devices` — Каталог устройств
- `POST /api/orders` — Заказ оборудования
- `GET /api/vod-lab/compare` — Сравнение устройств
