# VODeco ProjectHub Documentation

## Overview

**ProjectHub** — децентрализованная платформа для финансирования, разработки и коммерциализации инноваций в области водных технологий.

**Миссия:** Соединить исследователей, инвесторов и сообщество для решения глобального водного кризиса через открытую науку (DeSci) и децентрализованное финансирование.

---

## Core Principles (Основные Принципы)

### 1. Open Science (Открытая Наука)
- Все исследования публикуются с открытым доступом
- Peer review от экспертного сообщества
- Воспроизводимость экспериментов
- Прозрачное финансирование

### 2. Community-Driven Funding (Финансирование Сообществом)
- Проекты финансируются через DAO голосование
- Механизм квадратичного финансирования (quadratic funding)
- Вклад как токенами VOD, так и фиатом
- Прозрачное распределение средств

### 3. IP-NFT Integration (Интеллектуальная Собственность как NFT)
- Результаты исследований токенизируются как IP-NFT
- Доходы от лицензирования распределяются между инвесторами
- Возможность торговли долями интеллектуальной собственности
- Защита авторских прав через блокчейн

### 4. Milestone-Based Payouts (Выплаты по Этапам)
- Финансирование разблокируется по достижению milestones
- Автоматические выплаты через смарт-контракты
- Community vote для подтверждения выполнения этапов
- Возврат средств при неудаче проекта

## Project Lifecycle (Жизненный Цикл Проекта)

### Stage 1: Proposal Submission (Подготовка Предложения)
```
Исследователь/Команда
        ↓
Создание проектного предложения:
├── Название и описание проекта
├── Цели и ожидаемые результаты
├── Таймлайн и milestones
├── Бюджет и распределение средств
├── Команда и опыт
├── Технические спецификации
└── IP права и лицензирование
        ↓
Подача в ProjectHub
```

**Требования к предложению:**
- Минимум 500 слов описания
- Детальный бюджет с обоснованием
- Не менее 3 milestones
- Информация о команде (LinkedIn, портфолио)
- Предварительные данные/исследования (если есть)

### Stage 2: Community Review (Обзор Сообществом)
```
Проектное предложение
        ↓
[7-14 дней Public Review]
        ↓
Процесс:
├── Публикация на платформе
├── Обсуждение в Discord/Telegram
├── Peer review от экспертов
├── Вопросы и ответы (Q&A)
├── Правки по обратной связи
└── Финальная версия
        ↓
Статус: APPROVED / REJECTED / NEEDS_REVISION
```

**Механика ревью:**
- Каждый участник может оставить комментарий
- Эксперты имеют повышенный вес голоса
- Минимум 50 положительных голосов для перехода
- Критические замечания должны быть адресованы

### Stage 3: DAO Voting (Голосование DAO)
```
Утвержденное предложение
        ↓
[DAO Governance Vote]
        ↓
Параметры голосования:
├── Длительность: 7 дней
├── Quorum: 5% от staked VOD
├── Порог одобрения: 60%
├── Квадратичное голосование (1 VOD = 1 голос)
└── Возможность делегирования голоса
        ↓
Результат: FUNDED / NOT_FUNDED
```

**Условия финансирования:**
- Проект должен набрать минимум 60% голосов "за"
- Кворум должен быть достигнут
- Если одобрено — средства блокируются в смарт-контракте
- Если отклонено — возможна повторная подача через 30 дней

### Stage 4: Funding & Development (Финансирование и Разработка)
```
Одобренный проект
        ↓
[Smart Contract Escrow]
        ↓
Структура финансирования:
├── 20% — начальный капитал (после голосования)
├── 30% — после Milestone 1
├── 30% — после Milestone 2
└── 20% — после финального отчета
        ↓
Разработка:
├── Регулярные обновления (еженедельно)
├── Открытый репозиторий кода
├── Техническая документация
└── Взаимодействие с сообществом
```

**Требования к отчетности:**
- Еженедельные апдейты в блоге проекта
- Ежемесячные финансовые отчеты
- Открытый доступ к исходному коду
- Фото/видео документация прогресса

### Stage 5: Milestone Verification (Верификация Этапов)
```
Достижение Milestone
        ↓
[Submission of Evidence]
        ↓
Предоставление доказательств:
├── Технические отчеты
├── Результаты тестирования
├── Финансовая отчетность
├── Peer review от независимых экспертов
└── Демо/прототип
        ↓
[DAO Verification Vote]
        ↓
Результат:
├── VERIFIED → release next tranche
├── NEEDS_WORK → corrections required
└── FAILED → project termination
```

**Критерии верификации:**
- Соответствие техническим спецификациям
- Воспроизводимость результатов
- Экономическая целесообразность
- Отсутствие критических ошибок

### Stage 6: Commercialization (Коммерциализация)
```
Завершенный проект
        ↓
[IP-NFT Creation]
        ↓
Создание токенизированной интеллектуальной собственности:
├── Патенты и патентные заявки
├── Лицензии на использование
├── Права на коммерциализацию
└── Доли в будущих доходах
        ↓
[Revenue Distribution]
        ↓
Распределение доходов:
├── 40% — инвесторам проекта
├── 30% — команде разработки
├── 20% — DAO Treasury
└── 10% — community rewards
```

**Механика IP-NFT:**
- Каждый проект выпускает уникальный IP-NFT
- Держатели NFT получают дивиденды от лицензий
- Возможна торговля долями на二级市场
- Автоматическое распределение через смарт-контракты

## Legal Framework (Юридическая Рамка)

### IP Rights (Права ИС)
- До финансирования: IP принадлежит команде
- После финансирования: IP-NFT создается, доли распределяются
- Лицензии: обязательное лицензирование для коммерческого использования
- Open Source: компоненты могут быть open source по решению DAO

### Investment Terms (Условия Инвестирования)
- SAFE agreements для традиционных инвесторов
- Token warrants для VOD инвесторов
- Revenue sharing через смарт-контракты
- Exit clauses при невыполнении milestones

### Compliance (Соответствие)
- KYC/AML для инвесторов > $10K
- Securities law compliance
- Export control for dual-use technologies
- Environmental impact assessments

## FAQ

**Q: Кто может подать проект?**
A: Любой зарегистрированный пользователь с verified профилем.

**Q: Как долго длится процесс финансирования?**
A: От подачи до получения средств — 4-6 недель.

**Q: Что если проект не выполнит milestones?**
A: Оставшиеся средства возвращаются в DAO Treasury.

**Q: Как защищены инвесторы?**
A: Смарт-контракты escrow, milestone-based releases, IP-NFT collateral.

**Q: Можно ли инвестировать фиатом?**
A: Да, через партнерские платежные шлюсы с конвертацией в VOD.
